# Golang - Pemula sampai Mahir

Repositori ini adalah catatan pribadi saya untuk belajar Go (Golang) dari level pemula hingga mahir. Fokus utamanya adalah membangun kemampuan membuat aplikasi backend yang cepat, modern, scalable, serta relevan dengan cloud infrastructure.

**Tujuan Belajar**
- Mempelajari Go untuk membangun aplikasi backend yang cepat dan modern
- Menguasai konsep scalable app dan kebutuhan cloud infrastructure
- Memahami concurrency secara mendalam untuk kebutuhan sistem yang efisien

**Sumber Belajar Utama**
- Kursus Udemy Go oleh Programmer Zaman Now
- Dokumentasi resmi Go

**Target Pembaca**
- Diri saya sendiri sebagai dokumentasi proses belajar dan referensi cepat

**Topik yang Ingin Dicapai**
- Concurrency
- Web development dengan Go
- Testing
- Microservices dengan Go
- Scalable app

**Struktur Repository**
- `code` berisi contoh program, latihan, dan eksperimen
- `notes` berisi rangkuman materi dan catatan belajar
